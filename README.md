# Face recognition
## Prerequisites
- Python 3.x
- OpenCV library
    ```bash
    pip install opencv-python
    ```

## Description
The script opens your computerâ€™s camera and detects faces in real-time, drawing a green rectangle around each detected face. Press "a" to stop and close the video feed.

## How to Use
1. Run the script.
2. Allow camera access if prompted.
3. A window will open with your camera feed and detected faces marked in real-time.
4. Press "a" to close the feed.
   
## Troubleshooting
- Ensure your webcam is working properly.
- Verify the path for `haarcascade_frontalface_default.xml` in the script.
